# TripMe - Landing Page de Viagens

Projeto desenvolvido como desafio de HTML e CSS.

## Objetivo
Criar uma landing page de agência de viagens com navegação interna.

## Tecnologias
- HTML5
- CSS3

## Funcionalidades
- Banner com imagem e texto sobreposto
- Navegação interna com âncoras
- Seções informativas
- Efeitos hover com transição

## Estrutura
- index.html
- style.css
